# Capstone Projects

1) **Blog + Search (SSG + ISR + MDX)**
- MDX posts in content folder, `generateStaticParams`, `revalidate` for ISR.

2) **SaaS Dashboard (Auth + DB + Charts)**
- Auth.js, Prisma, protected server components, charts in client slots.

3) **Storefront (Products, Cart, Checkout)**
- Product listing (ISR), server actions for cart mutations, API for webhooks.

4) **Realtime Chat (Sockets + Actions)**
- WebSocket channel, optimistic updates in client; persist in server action.

5) **Multilingual Marketing Site (i18n + SEO)**
- Locale segments, OG images per locale, analytics + A/B via middleware.
